One Way Data Binding

String interpolation

With string interpolation, we use double-curly syntax. Just make sure the value we put in the curly braces is valid typescript code. All variables must also be present in the corresponding TypeScript file.

{{name}}

Event Binding

With event binding, we have an event (ex: click) and some sort of code to call when that event is triggered (ex: calling our alert function)

Property Binding

With property binding, we assign a property of one of our HTML elements using data from the TypeScript file

Just like with String interpolation, we have to make sure that the data/variable we're binding exists in the TypeScript file
