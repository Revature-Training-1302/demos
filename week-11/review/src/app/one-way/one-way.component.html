One Way Data Binding

Interpolation

With string interpolation, we use the double curly syntax. We can put any TypeScript code within the braces and it will be evaluated. If we want to use variables, they have to be present in the corresponding TypeScript file.

{{2+2}}

My name is {{name}}

Event Binding

With event binding, we can have an event (ex: click) and bind it to some function in our TS file. Ex: If we have press a button, an alert pops up.

Property Binding

With property binding, we assign a property of one of our HTML elements using data from te TypeScript file

Just like interpolation, we have to make sure that the variable actually exists in the TS file if we want to use it.

{{show}}

This will be hidden if show is set to false
